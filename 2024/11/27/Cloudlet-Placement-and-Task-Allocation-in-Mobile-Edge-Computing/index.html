  • 能耗包含两部分：静态能耗（计算资源节点保持待机状态所必须的能耗） + 执行任务能耗（和执行任务时间有关）
      • 表示的是计算资源节点的静态能耗，是保持开机的时间

Problem Definition

  • 对于 Cloudlet Placement and Task Allocation 问题（CPTA），可以被描述为：

    给定一个 AP 的集合，一个云端，一个用户提交的任务请求集合。CPTA 问题是要解决将 cloudlet 合理地放置在上，然后将用户的任务合理地分配给相应的 cloudlet，以达到总能耗最小，同时用户的任务延迟约束条件能够满足。
